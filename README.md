# simple miner game

## build
```sh
cc nob.c -o nob
./nob <linux|windows> [run]
```

## run
```sh
./build/main
```

## controls
### miner movement
```
    w
 a  s  d
```

### camera movement
```
    ↑
 ←  ↓  →
 
reset camera - r
```

### camera zoom 
```
numpad + 
numpad -
```

### save & load
```
save - k
load - l
```
